## 유지보수 관리 시스템

Laravel 12 기반의 유지보수 관리 시스템입니다.

## 📋 프로젝트 개요

- **프로젝트명**: 유지보수 관리 시스템
- **기술 스택**: Laravel 12, MySQL, WSL Ubuntu
- **개발 환경**: Docker, PHP 8.4
- **프로젝트 상태**: 개발 중

## 🚀 주요 기능

- **유지보수 요청 관리**: 클라이언트별 유지보수 요청 등록 및 관리
- **월간 보고서**: 월별 유지보수 현황 보고서 생성
- **공지사항**: 시스템 내 공지사항 관리
- **알림 시스템**: 실시간 알림 기능
- **파일 첨부**: 요청별 첨부파일 관리
